Marzipano

A new 360° media viewer

Why?

    We wanted a viewer which:

  • Is easier to customize using standard HTML, CSS and Javascript
  • Provides more control and can be extended
  • Can be integrated in a modern front-end development workflow
  • Supports all major browsers and devices

Marzipano Viewer and Tool

  • Marzipano Viewer: Library to view 360° media on the browser
  • Marzipano Tool: Browser application to create a simple tour that includes the Marzipano Viewer

Quick start guide

Quick start guide

  • Marzipano Tool processes panoramas and generates a basic tour application
  • Application can be customized and further developed by editing the HTML, CSS and Javascript code
  • Design principle: embrace web technologies

Marzipano Viewer

Media types

  • Cubic images
  • Multiresolution cubic images
  • Multiresolution flat images
  • Equirectangular images
  • Equirectangular videos

Media types

  • Any tile URL scheme can be used (Mapbox Demo)
  • Media can be loaded from different sources (not only URLs)
  • Images can be generated on the fly (Generated Cube Demo)
  • Marzipano be extended with custom media sources
  • Design principle: be modular and extensible

Browser support

    All major browsers and devices

  • WebGL (most modern browsers)
  • CSS 3D Transforms fallback (IE10, Safari 6-7, Androids without WebGL)
  • Flash fallback (IE8-9)

Hotspots

  • Hotspots are HTML elements
  • Can use CSS effects

  • Design principle: reuse when possible (but do not compromise on performance)

Performance

  • "Low Garbage" Javascript 
    var result = {};
    // writes to `result` object
    view.coordinatesToScreen({ yaw: 0, pitch: 0 }, result);
    console.log(result.x);
    console.log(result.y);
  • No large dependencies
  • <50KB gzipped

Developer-friendly

  • Uses standard technologies
  • Plays well with modern web development tools (e.g. browserify)
  • Does as little as possible and can be extended
  • Demos and docs at marzipano.net
  • Reference generated with JSDoc
  • Let us know if you have questions

Future work

  • More projections (e.g. stereographic for little planets)
  • More effects (e.g. motion blur)
  • Improvements to the Marzipano Tool
    • Alignment from EXIF data
    • Nadir image
    • Thumbnails
  • Experiments with depth maps and 3D
  • API improvements

Marzipano Tool

  • Processes panoramas on the browser
  • Requires Chrome or Firefox
  • Performance improving with new browser releases
  • Technology may be useful for other applications
